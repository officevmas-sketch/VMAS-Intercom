FULL PUSH SETUP - VMAS MOBILE INTERCOM

Your Web Push Certificate key is already added in firebase-config.js.

You only need to deploy the Firebase Function once from a laptop/desktop.

1) Install Node.js LTS from https://nodejs.org
2) Open Command Prompt / Terminal in this folder.
3) Run these commands one by one:

npm install -g firebase-tools
firebase login
firebase use vmas-intercom
cd functions
npm install
cd ..
firebase deploy --only functions,database

After deployment:
1) Open the app link on each mobile.
2) Login as each employee.
3) Tap Enable Notifications.
4) Allow notification permission.
5) On iPhone, install the PWA from Safari first: Share > Add to Home Screen.

Important limitations:
- Android: closed-app notifications work after permission is granted.
- iPhone: web push works only for Home Screen installed web apps and requires iOS 16.4 or later.
- Sound on lock-screen push depends on phone/browser settings. The in-app ringtone works when app is open.

Firebase Realtime Database rules for internal testing are included in database.rules.json.
